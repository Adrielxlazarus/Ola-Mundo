Olá, Mundo!

Na programação, "Olá, mundo" (ou "Hello, World") é a primeira frase tradicionalmente usada ao se aprender uma nova linguagem de programação

O objetivo é simples: mostrar como escrever, compilar (ou interpretar) e executar um programa básico que exibe uma mensagem na tela — geralmente essa mensagem é:
Olá, mundo!

Por que isso é importante?

Serve como teste básico para verificar se o ambiente de desenvolvimento está funcionando corretamente.

Ajuda a entender a estrutura mínima necessária para rodar um programa em determinada linguagem.

É um ritual de iniciação comum entre programadores, presente em livros, cursos e tutoriais.

Uma curiosidade interessante para Gustavo Guanabara, o "Olá, Mundo!" não é só uma mensagem, é o ritual de chegada ao mundo da programação, ele utiliza esse exemplo para mostrar:

- Como se estrutura um programa.

- Como configurar o ambiente de desenvolvimento.

- O processo básico de escrever, executar e ver o resultado
algo que se repete ao longo de qualquer jornada de aprendizado.
